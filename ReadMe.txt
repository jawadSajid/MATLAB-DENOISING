In this assignment, a function by the name of mySecondAssignment
is made where it expects an input image through command line as follows:

mySecondFunction('imagename')

The function is called and following actions are taken:
1- Salt and pepper noise is added.
2- 3x3 Averaging filter is applied manually.
3- Mean filtered image is stored.
4- 3x3 Median filter is applied manually.
5- Median filtered image is stored.
6- Input, noisy, mean filtered and median filtered images are shown.

help is also added which can be seen through typing:
"help mySecondAssignment" in the command line.
